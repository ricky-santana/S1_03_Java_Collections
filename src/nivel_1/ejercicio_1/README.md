# S1_03_Nivel_1_Ejercicio_1
**Duplicados**
## 📌 Enunciado
Crea una clase llamada Month con un atributo name, que almacene el nombre del mes. Añade 11 objetos Month a un ArrayList, dejando fuera "Agosto". 
Inserta este mes en la posición correspondiente y comprueba que el orden de los meses es correcto.

Convierte después el ArrayListen uno HashSet y verifica que no se permiten duplicados.

Por último, recorre la colección con un bucle for y con un Iterator.
